Flickr Fixr version

 Toggle features 

If this feature is enabled, Flickr Fixr will try to upscale photos when current display-versions are too small to fit the available space in your browser window. Also when the originally uploaded version of a photo is available in this situation (Should always be available when surfing your own photos), Flickr Fixr will use the original for best possible quality.

A note about up-scaling and photo notes: Sometimes when Flickr Fixr up-scales a photo, the size of the "notes canvas" does not up-scale similarly. When upscaling, an icon is shown above photo on the photopage:
If the icon is green, everything is alright, and you can work with notes directly on the up-scaled photo.
If the icon is orange and you want to work with notes, you should always first click the orange icon to align image size with size of the "notes canvas". Otherwise notes will not be correctly aligned.

Puts a column featuring albums on photographer's photostream. The 10 first albums as sorted in Flickr Organizer are shown in the column.

Adds an extra direct link in tags, pointing to the same photographer's similar tagged photos.

Adds two extra links in "album headers". One link pointing to a map with current album's photos, and another link pointing to the old hidden/forgotten page with album comments, which also is a great page for reading (or editing) long album descriptions.

When following map-link from a photopage, this ensures the photo you want to see the location of, is actually shown on the Flickr map!

As a bonus, also adds a link on photopage to show geoposition of photo directly on Google Maps.

Adds a pagination-bar when hovering mouse just in top of the photostreams, albumpages, favorite collections, ("beta experience") group pools and on the Explore page.

On the Explore page, add a link to the hidden/forgotten Explore Calendar and to a page showing some of the latest uploads to Flickr.

A little warning regarding the latest uploads though; Even though Safe Search is always activated on that page, you risk seeing stuff you might have preferred not to see.

Adds Tags, Collections and Map items to both the user ("You" dropdown) and the photographer menus (menu bar).

Via mouseover, show both Taken and Upload dates in detail when available. Taken date will also work as a link to more photos from the same photographer.

Notice, Flickr Fixr can only find the extended date & time information on public photos.

When a newsfeed is detected on a page, an orange feed-icon is added as link to this. The feed-icons are added in upper right corner of pages. Feeds are detected on the main flickr.com site, on status.flickr.net, and on the blogs on blog.flickr.net and code.flickr.net.

Notice, Flickr Fixr does nothing to handle the newsfeeds, it just makes them visible. If you are not using or familiar with newsfeeds, you might not have any reason to have this feature enabled.

When browsing a photostream in Date taken order, a subtle visual hint will be shown to warn you that latest added photos might not be on top of photostream.

If you have your Firefox webbrowser configured to open tabs in the background when ctrl-clicking links, you may have seen Flickr overriding this preference on search result pages (including the Trending Tags pages) and bring immediately focus to new tabs instead. This feature should prevent Flickr overriding your open tabs preferences. I have only seen the problem when using Firefox, so feature is default disabled for other webbrowsers.

Removes layer over images on "All sizes" pages, so you can use your browser's context menu for images.

I have been silent about this feature, because I don't want Flickr Fixr to be a tool for people who want to use other peoples photos without permission. But I have image tools like "Search Similar image" and "EXIF viewer" which I would like to access via the context menu.

Please respect ownership of images, and remember to ask permission for any public use of other peoples photos, if not already granted by an associated license.

You will have to reload any already open Flickr webpages to enable updated settings.
