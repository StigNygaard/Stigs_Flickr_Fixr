Congratulations!...

You have just installed or updated to Flickr Fixr version . If you have any open Flickr-pages already, you need to reload pages to make install/update take effect on these.

If you are new to Flickr Fixr, make sure to look through features on the Options Page.

WARNING of future minimum requirements!

Flickr Fixr will soon move (or do updates preparing the move) to new MV3 extension platform. A new more secure platform for browser-extensions being introduced for both Chromium and Firefox based browsers.

In the transition from the deprecated MV2 platform to the new MV3 platform, I will have to raise the minimum requirement for version of browser. If you like most are using a "normal" and auto-updating version of Firefox or Chromium (Chrome, Edge, Opera, Brave, etc), you will not be affected...

But if you are using a browser like for example Firefox ESR, Tor or Waterfox, there might be a period where you won't be able to update to the latest versions of Flickr Fixr. This could become a problem if Flickr makes site-changes being incompatible with current version of Flickr Fixr. If Flickr Fixr stops working for you or Flickr Fixr starts breaking Flickr functionality, I suggest to (report it to and) look for information and advice in the Flickr Fixr thread in Flickr Hacks forum:...
