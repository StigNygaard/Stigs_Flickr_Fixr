Congratulations!...

You have just installed or updated to Flickr Fixr version . If you have any open Flickr-pages already, you usually need to reload pages to make install/update take effect on these.

If you are new to Flickr Fixr, make sure to look through features on the Options Page.

What's new in this version?!...

  • After the recent change to dates, Flickr has made another - in my opinion bad for the user experience - change to the website.

    Modern webbrowsers per default makes it possible to resize multi-line input boxes. A great usability feature. But now Flickr has explicitly disabled that feature for comment-boxes on photopages, gallery pages and in group discussions.

    Flickr Fixr now "unlocks" these comment-boxes, bringing back the option to resize:

    [ Image placeholder - Flickr Resize ]

  • The "Extra top menu items" feature of Flickr Fixr was already adding links to Tags, Collections and Map in your top-menus. Now also a direct link to your Notifications page are being added to the "You" menu.

Remember you can enable and disable features on the Options Page if you don't like everything that Flickr Fixr can do. Some features also have extra options to tune the behavior.
