You have just installed or updated to Flickr Fixr version . If you have any open Flickr-pages already, you usually need to reload pages to make install/update take effect on these.

If you are new to Flickr Fixr, make sure to look through features on the Options Page.

10 years anniversary!

It was in Summer 2015 I published first version of Flickr Fixr. At that time I'm sure I didn't imagine it would still be alive and updated now, 10 years after. Sure, updates might be smaller and more sparse today, but there's still energy for a modest update now and then :-) New in this version is:

  • "Auto-expand height as you write" is now part of an updated "Resizable Comment boxes" feature. It uses an experimental browser-feature (css feature) currently implemented only in Chrome and Chromium based browsers. But my guess is, it will soon be webstandard and also implemented in Firefox and other browsers.
  • A re-designet Options Page. Should especially make it easier to control the "sub-settings" available for some features.

Remember, you can always enable and disable features on the Options Page if you don't like everything that Flickr Fixr can do.
