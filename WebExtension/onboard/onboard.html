Congratulations!...

You have just installed or updated to Flickr Fixr version . If you have any open Flickr-pages already, you usually need to reload pages to make install/update take effect on these.

If you are new to Flickr Fixr, make sure to look through features on the Options Page.

Fixed or improved in this version...

  • Fixed a "strange" bug that made some features break on flickr-accounts where no profile image had been set/uploaded.
  • Improved Flickr Fixr's photopage-features for non-pro Flickr users. Flickr's (relatively) new photopage as seen by non-pro users, looks a bit weird code-wise to me. And it is not very fun for me to "tweak" it. But I have at least made the support a bit better now (Still, the detailed timestamps via a mouseover is not visible for non-pro users and some layout details could be optimized). Not sure how much more energy I'm going to use on that page in general, hoping most Flickr Fixr users are Flickr Pro users anyway(?).

You can always enable and disable features on the Options Page if you don't like everything that Flickr Fixr can do. Some features also have extra settings to tune the functionality.
