Flickr Fixr version

 Toggle & customize features... 
Album-teaser column on photostreams

Puts a column featuring albums on photographer's photostream. The 10 first albums as sorted in Flickr Organizer are shown in the column.

Tag-links directly to same photographer's photos...

Adds an extra direct link in tags, pointing to the same photographer's similar tagged photos.

You can choose to see the extra link only when hovering a tag, or you can choose link to be shown persistent on all tags for a less "erratic" experience:

 

Links to album-map and album-comments

Adds two extra links in "album headers". One link pointing to a map with current album's photos, and another link pointing to the (very) old hidden/forgotten page with album comments, which also is a great page for reading (or editing) long album descriptions.

Improved map displaying for geotagged photos

1) Makes sure geoposition of photo is highlighted when following link to Flickr Map from a geotagged photo.
2) On the photopage for a geotagged photo, also adds a link to show geoposition of photo directly on Google Maps.

Photopage "Show in Photostream" links

Make it always possible to navigate from Photopage to the Photostream-page it belongs to.

Top-pagination for fast an easy navigation

Adds a pagination-bar when hovering mouse just in top of the photostreams, albumpages, favorite collections, ("beta experience") group pools and on the Explore page.

Explore Calendar and Recent Uploads

On the Explore page, add a link to the hidden/forgotten Explore Calendar and to a page showing some of the latest uploads to Flickr.

A little warning regarding the latest uploads though; Even though Safe Search is always activated on that page, you risk seeing stuff you might have preferred not to see.

Extra top menu items

Adds Tags, Collections, Map and Notifications items to the user menu ("You" dropdown) - and adds Tags, Collections and Map items to the photographer menus (menu bar).

Resizable Comment boxes... *

Allow comment-boxes to be resizable on photopages, gallerypages and in group discussions.

 

Auto-expand height as you write is now also implemented as part of this feature. However, it is using an experimental browser-feature which - as of April 2025 - is supported in Chrome/Chromium based browsers only. But support might also be coming to Firefox later.

Better date & time info on photopages

Make Taken date prominent displayed on page, rather than the Upload date.

Sometimes Flickr Fixr can also detect if a photo has been replaced after the first upload to Flickr, and the Replaced date will be shown on the photopage below the Upload and Taken dates.

Moving mouse over clock-icon next to the Upload and Taken dates will show both Taken and Upload time in full details. Taken date will also work as a link to more photos from the same photographer.

Note, these date/time features are optimized for wide browser windows and for the version of photopages which "Pro" Flickr users see. If you use a narrow browser window and/or doesn't have a "Pro" account, some things might not work or look exactly as shown and described here.

Show available newsfeeds

When a newsfeed is detected on a page, an orange feed-icon is added as link to this. The feed-icons are added in upper right corner of pages. Feeds are detected on the main flickr.com site, on status.flickr.net, and on the blogs on blog.flickr.net and code.flickr.net.

Notice, Flickr Fixr does nothing to handle the newsfeeds, it just makes them visible. If you are not using or familiar with newsfeeds, you might not have any reason to have this feature enabled.

Slideshow speed control...

Set the speed of Flickr slideshows. How many seconds do you want each image to be shown?

Collapsible Search-results sidebar

Makes sidebar on search-result pages collapsible. Look for "collapse button" next to the sidebar.

Normally the width of search-results should auto-adjust when toggling visibility of the sidebar. But if it doesn't do that automatically, making a tiny adjustment to width of browser window should trigger the search-results to "reflow". Or doing a shift-reload usually also reflows the page.

Warn if photostream ain't ordered by upload date

When browsing a photostream in Date taken order, a subtle visual hint will be shown to warn you that latest added photos might not be on top of the photostream.

Open in New Tab fix for Firefox

If you have your Firefox browser configured to open tabs in the background when ctrl-clicking links, you may have seen Flickr overriding this preference on search result pages (including the Trending Tags pages) and bring immediately focus to new tabs instead. This feature should prevent Flickr overriding your open tabs preferences. I have only seen the problem when using Firefox, so feature is default disabled for other webbrowsers.

Fix image context menu on "All sizes" pages

Removes layer over images on "All sizes" pages, to prevent missing or malfunctioning actions in browser's (right-click) context-menu.

Unless photos are Creative Commons licensed (or your own), Flickr usually adds an invisible layer above photos on the "All Sizes" pages, presumable to prevent image stealing.

I have never promoted this feature much, because I don't want Flickr Fixr to be a tool for people who want to use other people's photos without permission. But I have image tools like "Search Similar image" and "EXIF viewer" which I would like to be able to use via the browser's context menu.

Please respect ownership of images, and remember to ask permission for any non-private use of other peoples photos, if not already granted by an associated license.

Note, you will have to reload any already open Flickr webpages to make modified settings take effect on these.
